To open our project, these are the steps!
*make sure you are not running any other node server! it will not work properly otherwise! 

1. After the ZIP file is downloaded, open it up in VS Code. 
3. Make sure you have the "Live Server" extension installed. If not, install it in the "Extensions" button on the left bar in VS code. Right click on "\frontend\index.html" and click "Open with Live Server." 
2. Go to "Terminal" -> "New Terminal" -> "cd FairSplit" whatever the zip file name is when you downloaded it) -> "cd backend" -> "npm install" -> "node server.js". 


These steps should get the project running!
