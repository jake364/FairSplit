To open our project, these are the steps!

1. After the ZIP file is downloaded, open it up in VS Code. 
3. Make sure you have the "Live Server" extension installed. If not, install it in the "Extensions" button on the left bar in VS code. 
2. Go to "Terminal" -> "New Terminal" -> "cd FairSplit" whatever the zip file name is when you downloaded it) -> "cd backend" -> "npm install" -> "node server.js". 


These steps should get the project running!
